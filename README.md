# My Portfolio Website

This is my personal **portfolio website** built using **React** and **Tailwind CSS**. It showcases my skills, projects, and experience as a frontend web developer. The site is fully responsive, interactive, and built with modern web technologies.

## 🚀 Features

- Built with **React** for a fast and modern UI  
- Styled using **Tailwind CSS**  
- Smooth animations using **Framer Motion**  
- Projects, skills, and experience showcase  
- Fully responsive (mobile, tablet, desktop)  
- Clean and modern design  

## 🛠 Technologies Used

- React  
- Tailwind CSS  
- Framer Motion  
- React Router  
- Other modern libraries
